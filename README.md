# AI-Powered E-Commerce Website

Modern, responsive e-commerce platform with AI recommendations, built with React.js, Node.js, and MongoDB.

## Features
- 🛍️ Full e-commerce functionality
- 🤖 AI-powered product recommendations
- 💳 Razorpay payment integration
- 📱 Responsive design
- 🔐 Google & email authentication
- 👨‍💼 Admin dashboard
- 📦 Order tracking
- 🎨 Modern UI with animations

## Tech Stack
- **Frontend**: React.js, CSS3, React Router
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Authentication**: Firebase Auth
- **Payment**: Razorpay
- **Deployment**: Ready for Vercel/Netlify

## Quick Start
```bash
# Install dependencies
npm install

# Start development servers
npm run dev        # Frontend (port 3000)
npm run server     # Backend (port 5000)
```

## Environment Variables
Create `.env` files in both frontend and backend directories with required API keys.
